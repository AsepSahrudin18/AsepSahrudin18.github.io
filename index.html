About Me

Nama saya Asep Sahrudin. Saat ini menjadi mahasiswa di Sekolah Tinggi Teknologi Terpadu Nurul Fikri jurusan Teknik Informatika dari kelas weekend. Dengan posisi saat ini semester 4 dan saya sedang mencari pekerjaan di bidang Database Administrator atau bidang IT apa saja yang sesuai dengan kemampuan saya. Saya memiliki keterampilan HTML, CSS, JavaScript Fundamental, Linux Fundamental Operating System, familiar dengan Linux Directory Command Line dan familiar dengan SQL Qwery dll.

Saya memiliki pengalaman kerja part time di gym sebagai front office dengan deskripsi pekerjaan sebagai berikut : Menyiapkan semua jadwal kegiatan kelas dari pagi sampai selesai, membuat data laporan penjualan, menginput data user yang masuk dari pagi sampai jam kerja selesai, mengisi dalam dokumen perjanjian pengguna / anggota, membuat kartu member, melakukan filling system, konversi dokumen: kompres, unzip, merging dokumen, split dokumen, dan melakukan backup dokumen. Dan memeriksa data pengguna untuk setiap pengguna yang masuk dan keluar, serta memberikan terbaik pada pelanggan dengan senyum sapa dan salam setiap kali ada tamu yang masuk dan keluar.
